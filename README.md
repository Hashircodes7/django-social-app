# Django Social App

**Django Social App** is a backend-focused social media application built with Django, demonstrating core backend development concepts such as authentication, data modeling, and relational database management. This project emphasizes server-side logic, data integrity, and scalable design for real-world social platforms.

## Key Features

**Custom User Model & Profile**: Supports avatars and user bios.
**Authentication & Authorization**: Secure user registration, login, and logout with password hashing.
**Post Management**: Users can create, update, delete, and view posts with optional image uploads.
**Comments & Likes**: Supports commenting on posts and liking posts or comments.
**Follow System**: Users can follow or unfollow other users.
**Admin Management**: Full control over users, posts, comments, likes, and followers through Django Admin.
**Data Integrity & Validation**: Enforced through model constraints and form validation.
**Signal Handling**: Automatic profile creation on new user registration.

## Tech Stack

**Backend**: Python, Django
**Database**: PostgreSQL
**Frontend**: HTML/CSS templates
**Media Management**: Image uploads for posts and user avatars

## Purpose

This project is intended as a backend developer showcase, demonstrating structured server-side logic, proper use of ORM, class-based views, signals, and user authentication. It provides a solid foundation for building scalable social media or content-driven applications.
